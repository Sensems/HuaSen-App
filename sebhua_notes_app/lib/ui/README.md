# lib/ui

Shared UI layer: reusable components and theme definitions used across
all feature modules.

## Contents

- **components/** - Reusable widgets (buttons, cards, dialogs, inputs)
- **theme/** - App-wide ThemeData, color tokens, typography scale, spacing

Components here are generic and feature-agnostic. Feature-specific widgets
live inside their respective `lib/features/<feature>/presentation/` folders.
